# PolTect

PolTect is an AI-based Chrome Extension that detects the bias in news articles. The file for Pretrained model can be found [here](https://drive.google.com/file/d/1zQCVDXIOm_j38dZZi2oAxScxLwGcW8y1/view?usp=sharing). Extract the .zip file and run the app.py file before using the extension locally.
